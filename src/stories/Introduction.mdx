# Getting Started with MDX

이것은 **MDX를 사용한 Storybook 문서**의 예시입니다!

## MDX란?

MDX는 Markdown 문법 안에서 JSX 컴포넌트를 사용할 수 있게 해주는 포맷입니다.

## 기본 사용법

### 1. 텍스트 서식

- **볼드 텍스트**
- *이탤릭 텍스트*
- `코드 블록`

### 2. 코드 블록

```javascript
const greeting = "안녕하세요!";
console.log(greeting);
```

### 3. React 컴포넌트 임베딩

컴포넌트를 직접 문서에 포함할 수 있습니다:

```jsx
export const Welcome = () => (
  <div style={{ 
    padding: '20px', 
    background: '#f0f0f0', 
    borderRadius: '8px',
    margin: '20px 0'
  }}>
    <h3>Welcome!</h3>
    <p>이것은 MDX 안에서 렌더링된 React 컴포넌트입니다.</p>
  </div>
);

<Welcome />
```

### 4. 인터랙티브 요소

export const Counter = () => {
  const [count, setCount] = React.useState(0);
  
  return (
    <div style={{ textAlign: 'center', margin: '20px' }}>
      <button 
        onClick={() => setCount(count + 1)}
        style={{
          padding: '10px 20px',
          fontSize: '16px',
          backgroundColor: '#007bff',
          color: 'white',
          border: 'none',
          borderRadius: '5px',
          cursor: 'pointer'
        }}
      >
        클릭 수: {count}
      </button>
    </div>
  );
};

<Counter />

## 더 많은 기능들

MDX를 통해 다음과 같은 것들이 가능합니다:

1. **컴포넌트 문서화** - Props와 사용법을 직관적으로 설명
2. **인터랙티브 예제** - 실제 동작하는 컴포넌트를 바로 보여주기
3. **디자인 시스템 가이드** - 스타일 가이드와 컴포넌트를 함께 문서화
4. **튜토리얼 작성** - 단계별 가이드를 컴포넌트와 함께 제공
